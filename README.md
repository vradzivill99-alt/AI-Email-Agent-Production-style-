# AI Email Processing Agent

## 🚀 Overview
Production-style AI system for automating email processing.

## Features
- Email classification (client / spam / internal)
- Priority detection
- AI-generated replies
- REST API

## Tech Stack
- Python
- FastAPI
- OpenAI API

## API

POST /process-email

Example:
{
  "subject": "Partnership request",
  "body": "We want to collaborate..."
}

## Impact
- Reduces manual email handling by ~70%
- Improves response time
